
# For loop
```go
for i := 0; i < 10; i++{
	// Code
}
```

# While Loop
```go
for i < 10 {
	// Code
}

// While true
for {
	// Code
}

```

# Range
```go

// Iterate over a collection
for index, value:= range collectionName {
	// Code
	// Note* index and value are omittable via '_' character
	}

```
